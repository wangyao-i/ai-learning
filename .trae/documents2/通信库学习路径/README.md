## 核心特性

### 1. 集合通信原语
- **AllReduce**：所有进程归约并广播结果
- **AllGather**：聚集所有进程数据到每个进程
- **Broadcast**：从根进程广播到所有进程
- **ReduceScatter**：归约后分散到各进程
- **AlltoAll**：全对全数据交换

### 2. 通信算法
- **Ring**：环拓扑通信，线性复杂度
- **Mesh**：网格拓扑，适用于单机多卡
- **RHD**：递归二分倍增，对数复杂度
- **PairWise**：逐对通信，避免一打多
- **Star**：星型拓扑，适用于有根操作

### 3. 性能优化
- **流水线并行**：并发使用 Server 内与 Server 间链路
- **拓扑感知**：自动发现最优通信路径
- **动态算法选择**：根据数据量和拓扑选择最优算法
- **内存优化**：高效的 Buffer 管理

### 4. 部署特性
- **多链路支持**：HCCS、RoCE、PCIe
- **单算子/图模式**：支持两种执行模式
- **多框架适配**：PyTorch、MindSpore 等
- **分布式部署**：支持多机多卡
