# HCCL 快速参考指南

## 1. 快速开始

### 1.1 最简示例

```cpp
#include "hccl/hccl.h"

int main() {
    // 初始化
    HcclComm comm;
    HcclCommInitAll(&comm, 8);
    
    // 通信
    HcclAllReduce(sendBuf, recvBuf, count, dataType, op, comm, stream);
    
    // 清理
    HcclCommDestroy(comm);
    return 0;
}
```

### 1.2 标准流程

```
初始化设备 → 初始化通信域 → 分配内存 → 创建流 → 执行通信 → 同步 → 清理
```

### 2.2 通信算子

```cpp
// AllReduce - 梯度聚合
HcclAllReduce(sendBuf, recvBuf, count, dataType, op, comm, stream);

// AllGather - 数据收集
HcclAllGather(sendBuf, recvBuf, sendCount, dataType, comm, stream);

// Broadcast - 参数分发
HcclBroadcast(sendBuf, recvBuf, count, dataType, root, comm, stream);

// ReduceScatter - 归约分散
HcclReduceScatter(sendBuf, recvBuf, recvCount, dataType, op, comm, stream);
```

## 7. 性能测试速查

### 7.1 基准测试

```cpp
#include <chrono>

auto start = std::chrono::high_resolution_clock::now();
HcclAllReduce(...);
aclrtSynchronizeStream(stream);
auto end = std::chrono::high_resolution_clock::now();

double time = std::chrono::duration<double>(end - start).count();
double bandwidth = dataSize / time / 1e9;  // GB/s

std::cout << "Time: " << time * 1000 << "ms" << std::endl;
std::cout << "Bandwidth: " << bandwidth << " GB/s" << std::endl;
```

### 7.2 性能指标

| 指标 | 说明 | 优化方法 |
|------|------|----------|
| 延迟 | 单次通信时间 | 使用Star算法 |
| 带宽 | 数据传输速率 | 使用Mesh/RHD算法 |
| 吞吐量 | 单位时间通信量 | 启用流水线 |

## 8. 代码模板

### 8.1 完整模板

```cpp
#include "hccl/hccl.h"
#include <vector>

int main() {
    // 1. 初始化设备
    int deviceCount;
    aclrtGetDeviceCount(&deviceCount);
    aclrtSetDevice(0);
    
    // 2. 初始化通信域
    HcclComm comm;
    HcclCommInitAll(&comm, deviceCount);
    
    // 3. 获取rank信息
    int rank, size;
    HcclGetRank(comm, &rank);
    HcclGetSize(comm, &size);
    
    // 4. 分配内存
    size_t dataSize = 1024 * 1024;
    void* sendBuf = AllocDeviceMem(dataSize);
    void* recvBuf = AllocDeviceMem(dataSize);
    
    // 5. 创建流
    aclrtStream stream;
    aclrtCreateStream(&stream);
    
    // 6. 执行通信
    HcclResult result = HcclAllReduce(
        sendBuf, recvBuf, dataSize,
        HCCL_DATA_TYPE_FLOAT32,
        HCCL_REDUCE_SUM, comm, stream
    );
    
    if (result != HCCL_SUCCESS) {
        std::cerr << "Error: " << result << std::endl;
        return -1;
    }
    
    // 7. 同步等待
    aclrtSynchronizeStream(stream);
    
    // 8. 清理资源
    aclrtDestroyStream(stream);
    FreeDeviceMem(sendBuf);
    FreeDeviceMem(recvBuf);
    HcclCommDestroy(comm);
    
    return 0;
}
```

### 8.2 分布式训练模板

```cpp
class DistributedTrainer {
public:
    DistributedTrainer(int deviceCount) {
        HcclCommInitAll(&comm_, deviceCount);
        HcclGetRank(comm_, &rank_);
        HcclGetSize(comm_, &size_);
        aclrtCreateStream(&stream_);
    }
    
    ~DistributedTrainer() {
        aclrtDestroyStream(stream_);
        HcclCommDestroy(comm_);
    }
    
    void AggregateGradients(std::vector<float>& gradients) {
        HcclAllReduce(
            gradients.data(),
            gradients.data(),
            gradients.size(),
            HCCL_DATA_TYPE_FLOAT32,
            HCCL_REDUCE_SUM,
            comm_,
            stream_
        );
        aclrtSynchronizeStream(stream_);
        
        // 平均梯度
        float scale = 1.0f / size_;
        ScaleGradients(gradients, scale);
    }
    
private:
    HcclComm comm_;
    aclrtStream stream_;
    int rank_;
    int size_;
};
```

## 11. 快速决策树

```
需要通信？
  ├─ 梯度聚合 → AllReduce
  ├─ 参数收集 → AllGather
  ├─ 参数分发 → Broadcast
  └─ 梯度分片 → ReduceScatter

数据量？
  ├─ <64KB → Star算法
  ├─ <1MB → Ring算法
  └─ ≥1MB → Mesh/RHD算法

遇到问题？
  ├─ 超时 → 检查网络
  ├─ 内存不足 → 减小Buffer
  └─ 性能差 → 优化算法
```
