### 2. е…ій”®жӯҘйӘӨиҜҰи§Ј

#### жӯҘйӘӨ1: еқ—зҙўеј•и§Јжһҗ
ж №жҚ®`block_tables`и§ЈжһҗжҜҸдёӘеәҸеҲ—дҪҝз”Ёзҡ„KVеқ—зҙўеј•пјҢе°ҶйҖ»иҫ‘дёҠиҝһз»ӯзҡ„еәҸеҲ—жҳ е°„еҲ°зү©зҗҶдёҠзҡ„еқ—гҖӮ

#### жӯҘйӘӨ2: KVзј“еӯҳжЈҖзҙў
дҪҝз”Ёи§ЈжһҗеҮәзҡ„еқ—зҙўеј•д»Һ`key_cache`е’Ң`value_cache`дёӯжЈҖзҙўеҜ№еә”зҡ„й”®еҖјеҜ№гҖӮ

#### жӯҘйӘӨ3: жіЁж„ҸеҠӣи®Ўз®—
и®Ўз®—жҹҘиҜўеҗ‘йҮҸдёҺй”®еҗ‘йҮҸзҡ„зӮ№з§ҜпјҢеҫ—еҲ°жіЁж„ҸеҠӣеҲҶж•°пјҡ
```python
attn_scores = torch.matmul(query, key.transpose(-2, -1)) / math.sqrt(head_dim)
```

#### жӯҘйӘӨ4: жҺ©з Ғеә”з”Ё
ж №жҚ®`context_lens`з”ҹжҲҗжіЁж„ҸеҠӣжҺ©з ҒпјҢйҳІжӯўе…іжіЁи¶…еҮәдёҠдёӢж–Үй•ҝеәҰзҡ„tokenпјҡ
```python
mask = torch.arange(seq_len) < context_len.view(-1, 1)
attn_scores = attn_scores.masked_fill(~mask, -1e9)
```

#### жӯҘйӘӨ5: еҠ жқғжұӮе’Ң
еҜ№жіЁж„ҸеҠӣеҲҶж•°иҝӣиЎҢsoftmaxеҪ’дёҖеҢ–пјҢз„¶еҗҺдёҺеҖјеҗ‘йҮҸиҝӣиЎҢеҠ жқғжұӮе’Ңпјҡ
```python
attn_weights = torch.softmax(attn_scores, dim=-1)
output = torch.matmul(attn_weights, value)
```

## еӣӣгҖҒPagedAttentionзҡ„дјҳеҠҝ

1. **еҶ…еӯҳж•ҲзҺҮ**: йҖҡиҝҮеқ—зә§з®ЎзҗҶеҮҸе°‘еҶ…еӯҳзўҺзүҮ
2. **зҒөжҙ»жү©еұ•**: ж”ҜжҢҒеҸҳй•ҝеәҸеҲ—зҡ„й«ҳж•ҲеӨ„зҗҶ
3. **жү№йҮҸдјҳеҢ–**: ж”ҜжҢҒеӨҡдёӘеәҸеҲ—е…ұдә«KVзј“еӯҳжұ 
4. **й«ҳжҖ§иғҪ**: еҮҸе°‘еҶ…еӯҳеҲҶй…Қе’Ңж•°жҚ®жӢ·иҙқејҖй”Җ

## дә”гҖҒжҖ»з»“

PagedAttentionеҮҪж•°зҡ„дә”дёӘж ёеҝғеҸҳйҮҸеҚҸеҗҢе·ҘдҪңпјҢе®һзҺ°дәҶй«ҳж•Ҳзҡ„KV Cacheз®ЎзҗҶе’ҢжіЁж„ҸеҠӣи®Ўз®—пјҡ

| еҸҳйҮҸ | з»ҙеәҰ | ж ёеҝғдҪңз”Ё |
|------|------|----------|
| `query` | (num_tokens, num_heads, head_dim) | еҪ“еүҚtokenзҡ„жҹҘиҜўеҗ‘йҮҸ |
| `key_cache` | (num_blocks, block_size, num_kv_heads, head_dim) | йў„и®Ўз®—зҡ„й”®еҗ‘йҮҸзј“еӯҳжұ  |
| `value_cache` | (num_blocks, block_size, num_kv_heads, head_dim) | йў„и®Ўз®—зҡ„еҖјеҗ‘йҮҸзј“еӯҳжұ  |
| `block_tables` | (num_seqs, max_num_blocks_per_seq) | еәҸеҲ—еҲ°KVеқ—зҡ„жҳ е°„иЎЁ |
| `context_lens` | (num_seqs,) | жҜҸдёӘеәҸеҲ—зҡ„жңүж•ҲдёҠдёӢж–Үй•ҝеәҰ |

йҖҡиҝҮиҝҷдәӣеҸҳйҮҸзҡ„еҚҸеҗҢе·ҘдҪңпјҢPagedAttentionе®һзҺ°дәҶжҜ”дј з»ҹжіЁж„ҸеҠӣжңәеҲ¶жӣҙй«ҳзҡ„еҶ…еӯҳж•ҲзҺҮе’Ңи®Ўз®—жҖ§иғҪпјҢжҳҜvLLMиғҪеӨҹй«ҳж•ҲеӨ„зҗҶеӨ§и§„жЁЎиҜӯиЁҖжЁЎеһӢжҺЁзҗҶзҡ„е…ій”®жҠҖжңҜд№ӢдёҖгҖӮ
