## 📝 学习内容速览

### 1. Self-Attention四步
1. **线性变换**: Q = X @ W_q, K = X @ W_k, V = X @ W_v
2. **计算分数**: Scores = Q @ K^T / sqrt(d_k)
3. **归一化**: Weights = softmax(Scores)
4. **加权求和**: Output = Weights @ V

### 2. 关键概念
- **Q (Query)**: 查询向量
- **K (Key)**: 键向量  
- **V (Value)**: 值向量
- **除以sqrt(d_k)**: 防止梯度消失

### 3. Multi-Head Attention
多个注意力头并行计算，捕捉不同依赖关系
