## 📋 学习内容概要

### 1. Self-Attention计算过程
**四步计算流程**:
1. **线性变换**: Q = X @ W_q, K = X @ W_k, V = X @ W_v
2. **计算注意力分数**: Scores = Q @ K^T / sqrt(d_k)
3. **Softmax归一化**: Weights = softmax(Scores)
4. **加权求和**: Output = Weights @ V

### 2. Multi-Head Attention
```
MultiHead(Q, K, V) = Concat(head_1, ..., head_h) @ W_o
其中 head_i = Attention(Q @ W_q_i, K @ W_k_i, V @ W_v_i)
```

### 3. 关键知识点
- **Q/K/V含义**: Query(查询)、Key(键)、Value(值)
- **除以sqrt(d_k)**: 防止点积过大导致softmax梯度消失
- **多头优势**: 每个头关注不同的依赖关系
- **计算复杂度**: O(n²·d)，其中n是序列长度，d是维度
